# MPlayer

**MPlayer** is a modern, feature-rich Android music player built entirely in **Dart** using **Flutter**. It is designed to be a pixel-perfect recreation of the "PixelPlayer" experience, featuring a stunning **Material 3 Expressive** UI.

## Features (Planned)
-   **100% Dart Architecture**: Built with Riverpod, Drift, and Retrofit.
-   **Local Music Playback**: Scans device for audio files.
-   **Expressive UI**: Dynamic theming, motion-rich interactions, and waveform visualizations.
-   **AI Playlists**: Integrated with Gemini AI for generating playlists from prompts.
-   **Local Casting**: Built-in HTTP server to cast local files to other devices.

## Tech Stack
-   **Language**: Dart 3.x
-   **Framework**: Flutter
-   **State Management**: `flutter_riverpod`
-   **Database**: `drift` (SQLite)
-   **Networking**: `retrofit` + `dio`
-   **Audio**: `just_audio` + `audio_service`
